== Description ==

WB FAQ is a custom post type based FAQs showing plugin.WB FAQ plugin works using shortcodes [wb-faq].
Also can be add in themes PHP template  using  <?php echo do_shortcode('[wb-faq]'); ?>
You can display FAQ list anywhere in your site using this shortcode . It is responsive and easy to use.
